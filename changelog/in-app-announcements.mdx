</Note>

Autter now surfaces new product updates and changelog entries directly in the dashboard — no need to check this page manually or wait for an email.

## What's new

When a new feature, improvement, or fix is released, a notification badge appears in the Autter dashboard. Clicking it opens the full announcement so you can read the update in context, right where you work.

## Highlights

- **Instant visibility** — Updates appear in your dashboard the moment they're published
- **In-context** — Read about new features without leaving the Autter UI
- **Unread tracking** — The badge shows how many updates you haven't seen yet, and clears once you've opened them

## How it works

1. A notification badge appears in the Autter dashboard when a new update is available
2. Click the badge to open the announcement panel
3. Read the full update — release notes, new features, and fixes
4. The badge clears once you've viewed the announcement

No configuration required. In-app announcements are enabled for all Autter accounts automatically.
