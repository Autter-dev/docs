# Autter documentation

This repository contains the Mintlify documentation for the Autter platform and open source CLI.

## Local development

Install the Mintlify CLI:

```bash
npm install --global mint
```

Start the preview from this directory:

```bash
mint dev
```

Open `http://localhost:3000`.

## Validate changes

```bash
mint broken-links
mint validate
mint a11y
```

## Content sources

- Use [autter.dev](https://autter.dev) for product and brand information.
- Use [Autter-dev/autter-cli](https://github.com/Autter-dev/autter-cli) for CLI behavior and commands.
